# Used Car Sales Dashboard (Power BI)

This project is a Power BI dashboard created to analyze used car sales data.

## Dashboard Features
- Total number of cars
- Average selling price by fuel type
- Total cars by transmission
- Interactive filters using slicers (Year, Fuel)

## Tools Used
- Power BI Desktop
- GitHub

## How to View the Dashboard
Download the .pbix file and open it using Power BI Desktop.
